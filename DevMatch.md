# DevMatch 🚀

DevMatch is a project I built to help students find teammates and work on projects together.

In college, many students have ideas but they don’t find the right people, so I created this platform where:

* A student can create a project
* Other students can apply
* Teams can be formed easily

---

## Features

* Login and Signup system
* Create, edit, delete projects
* Apply to projects
* Accept / Reject system
* Dashboard for managing requests
* Students page with filters

---

## Special Logic

* Cannot apply to your own project
* Maximum 3 attempts allowed
* Reapply only after rejection
* Dashboard shows only latest applications

---

## Tech Stack

* Node.js
* Express.js
* MongoDB
* EJS

---

## How to Run

```
npm install
npm start
```
